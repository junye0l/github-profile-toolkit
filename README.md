# GitHub Profile Toolkit

깃허브 프로필을 더 쉽게 꾸밀 수 있도록 도와주는 GitHub 프로필 커스터마이징 툴킷입니다. 

A toolkit for easily customizing your GitHub profile with badges, typing banners, GitHub stats, Pokémon Commit Card, and markdown snippets.

---

## Features | 주요 기능

- 기술 스택 뱃지 생성 | Generate tech stack badges
- 타이핑 배너 생성 | Create typing banner messages
- GitHub 통계 카드 추가 | Add GitHub stats cards
- 포켓몬 카드 | Pokémon Commit Card
- 마크다운 코드 즉시 복사 | Copy markdown instantly
- 실시간 미리보기 지원 | Live preview support
- 직관적인 UI 제공 | Simple and intuitive UI

---

## Why I Built This | 제작 이유

GitHub 프로필을 꾸미면서 여러 사이트를 오가며 뱃지, 통계 카드, 마크다운 코드를 하나씩 생성해야 하는 과정이 번거롭게 느껴졌습니다.

자주 사용하는 GitHub 프로필 커스터마이징 기능들을 한 곳에서 더 편하게 사용할 수 있도록 제작했습니다.

추후에는 다양한 위젯과 편의 기능들을 계속 확장해 나갈 예정입니다.

</br>

While customizing my GitHub profile, I found it tedious to repeatedly visit multiple websites just to generate badges, stats cards, and markdown snippets.

So I built a small tool that combines commonly used GitHub profile customization features into one place with a simpler workflow.

More widgets and customization features will be added in the future.

---

## How It Works | 사용 방법

1. 원하는 기능 선택 | Select the feature you want
2. 스타일 및 옵션 설정 | Customize styles and options
3. 생성된 마크다운 코드 복사 | Copy the generated markdown
4. GitHub 프로필 README에 붙여넣기 | Paste it into your GitHub profile README

---

## Supported Features | 지원 기능

### Tech Stack Badges

기술 스택 뱃지를 생성할 수 있습니다.

Generate customizable technology stack badges for your GitHub profile.

---

### Typing Banner

타이핑 애니메이션 배너를 생성할 수 있습니다.

Create animated typing text banners for your README.

---

### GitHub Stats

GitHub 통계 카드 및 프로필 위젯을 추가할 수 있습니다.

Add GitHub stats cards and profile widgets.

---

### Pokémon Commit Card

GitHub 커밋 통계를 기반으로 포켓몬 카드를 생성할 수 있습니다.

커밋 활동에 따라 포켓몬 정보와 카드가 동적으로 변경됩니다.

Generate Pokémon-style profile cards based on your GitHub commit statistics.

The Pokémon card changes dynamically depending on your GitHub activity and commit data.
